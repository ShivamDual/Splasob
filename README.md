# Splasob
Splasob is a mobile-friendly web game where players swipe to smash cubes and earn scores. Built with HTML, CSS, and JavaScript, it uses JSON for game data and includes images, music, and other assets. Multiple modes, dynamic backgrounds, and responsive design make it fun and interactive.
